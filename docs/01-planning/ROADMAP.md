## 🔄 반복 개발 사이클

각 Phase 내에서:

```
[기획] → [개발] → [테스트] → [피드백] → [개선]
   ↑                                      │
   └──────────────────────────────────────┘
```

1. **기획 (1일)**: 기능 상세 명세 작성
2. **개발 (5-7일)**: 실제 구현
3. **테스트 (1-2일)**: 직접 사용해보며 버그 발견
4. **피드백 (1일)**: 개선점 정리
5. **개선 (2-3일)**: 피드백 반영

→ 다음 기능으로 이동
